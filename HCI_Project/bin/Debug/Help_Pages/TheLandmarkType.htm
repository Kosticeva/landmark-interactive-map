All about the Landmark Types

Every landmark has a landmark type. Landmark type is represented by his:

  • ID,
  • Name,
  • Description and
  • Picture

ID

Every landmark type is identified with its ID. The landmark type ID is entered at the topmost text field. The ID should be unique for your landmark type. If you enter an ID that is already the ID of some other landmark type, you will be informed of that when you try to save your landmark.

Name

Every landmark type has to have a name. The name is not unique, and can be in any form you like. It is entered at the second text field from the top.

Description

You can write anything you want about your landmark type in its description. The description is entered at the third field from the top.

Landmark type picture

Every landmark type needs to have a picture. If you choose a landmark type for your landmark, but do not choose the picture for the landmark, the landmark type picture will be placed as the landmark picture. You can choose a picture for your type by clicking the 'Browse' button. Picture you chose will appear on the left side of the 'Browse' button.
