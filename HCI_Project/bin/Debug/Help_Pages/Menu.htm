All about the Menu

All the actions you can perform on the map are placed in the menu.

The menu contains 4 sections:

  1. File
  2. Edit
  3. View
  4. Help

File

File section contains 3 items: New, Save and Exit.

New section has 3 options:

  1. New Landmark - opens dialog for new landmark
  2. New Landmark Type - opens list of all created landmark types
  3. New Tag - opens list of all created tags

You can use keyboard keys or toolbar options to access each of the New section items. Control+L opens the New Landmark dialog (), Control+Y opens the New Landmark Type dialog () and Control+T opens the New Tag dialog ().

Save option saves all the data you created. It can be accessed through the menu or pressing the Control+S keys.

Exit option is used to exit the application. When you select this option, a dialog will appear and you will need to confirm you want to exit. If you confirm and you haven't saved your data, another dialog will appear, this time asking you if you would like to save your data. If you confirm, your data will be saved and you will exit the application. If you cancel, your data won't be saved and you will exit the application. You can access this option through the menu or pressing the Alt+F4 keys.

Edit

Exit section contains 5 items: Cut, Copy, Paste, Delete and Find...

Cut, Copy, Paste and Delete are controls used for manipulating the placed landmark.

  • Cut option removes the landmark from the map but saves its copy on the clipoard. It can be accessed through the menu, pressing the Control+X keys or through the toolbar option ().
  • Copy option saves the copy of the landmark on the clipboard, but leaves the original landmark on the map. It can be accessed through the menu, pressing the Control+C keys or through the toolbar option ().
  • Paste option places the copy of the landmark at the selected position on the map. You can paste landmarks that you have copied with Copy or Cut option. It can be accessed through the menu, pressing the Control+V keys or through the toolbar option ().
  • Delete option removes the landmark from the map. It is not copied on the clipboard, but if you have copied it with Copy or Cut options, you can place it again on the map. This option only deletes landmark image from the map - it doesn't delete the landmark from the database. This option can be accessed through the menu, pressing the Delete key or through the toolbar option ().

To use the Cut, Copy, Paste and Delete command, you must first select the landmark you want to manipulate.

Find... option opens a dialog with all created landmarks from which you can edit, delete or place landmarks on the map. You can also find any landmark you want from this dialog. Find... option can be opened from the menu, pressing the Control+F keys or through the toolbar option ()

View

View section has 2 items: Change Map and Toolbar...

Change Map option opens a dialog where you can select background for your map. The selection is not remembered, so the next time you log in with your account, the background map will be the default one.

Toolbar... option opens a menu with one check-box marked 'Visible', and two radio butons marked 'Horizontal' and 'Vertical'. Checking the 'Visible' check-box places the toolbar on the map. It is placed in the upper left corner of the map. It contains 8 options: 3 for creating landmark/landmark type/tag, and 5 from the 'Edit' section, used for landmark manipulation and searching for landmark.

Selecting the radio buttons will change toolbar's orientation. 'Horizontal' will change toolbar's orientation to horizontal, if toolbar's orientation until then was vertical and 'Vertical' will change toolbar's orientation from horizontal to vertical. You can always hide the toolbar by unchecking the 'Visible' check-box.

You can also move the toolbar around the map.

Help

The Help section opens the index page of The Natural Heritage Map helper. It can be accessed through the menu or by pressing the Control+H keys.
