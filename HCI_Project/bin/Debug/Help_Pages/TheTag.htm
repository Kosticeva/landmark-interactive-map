All about the Tag

Every landmark can have none, one or multiple tags attached on it. Each tag is represented by:

  • ID,
  • Description and
  • Color

ID

Every tag is identified with its ID. The tag ID is entered at the topmost text field. The ID should be unique for your tag. If you enter an ID that is already the ID of some other tag, you will be informed of that when you try to save your tag.

Description

You can write anything you want about your tag in its description. The description is entered at the second field from the top.

Color

You can choose a color for your tag from the drop down list below the description field. The list contains a color map, where you can choose the color by selecting the color from the slider on the right and clicking on the right shade in the color map. You can also manually input decimal values of Red, Green and Blue component of your color, or you can choose an existing color by clicking the 'Standard' button at the bottom of the list.

'Standard' button opens a list of available colors, where you can choose a color by clicking on it. When you choose a color, its name or its hexadecimal value will appear at the top of the list.
